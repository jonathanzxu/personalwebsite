Real-time motion capture-based robotic arm control

Summer Research Intern Project completed at California Baptist Unversity in Summer 2021 under Dr. Seung-Jae Kim. I was the sole contributor to this project. Involves using the PhaseSpace 3D motion capture system to extract 3D coordinates from infrared markers in real time. Interfaces with the InterbotiX ReactorX 150 Robotic Arm to mimic gestures in real-time modeled by markers attached to a human arm. Programmed fully in Python. Details on mathematical calculations, API interaction with PhaseSpace, and serial USB communication protocol with robotic arm provided in official documentation. Code is open-source under MIT license. Research applied to visual distortion stroke recovery and physical therapy research.
